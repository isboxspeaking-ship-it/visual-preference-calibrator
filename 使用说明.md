# 自然流｜该项目品牌视觉校准器 Demo

## 打开方式

直接双击 `index.html` 即可在浏览器打开。图片位于 `assets` 文件夹，移动时需要将整个文件夹一起移动。

## 当前功能

- 使用自然流品牌视觉系统：本白底、青蓝色系、iP标志与顶部条纹；
- 完整整合D2飞书文档中的项目理解、现有资产、现状诊断、升级目标、品牌内核与三方向说明；
- 三个方向共47张参考图：A智能系统型16张、B成长陪伴型16张、C创始人科技型15张；
- 手机与电脑自适应；
- 左滑喜欢、右滑不喜欢、上滑不确定；
- 同时提供按钮，避免客户记错手势；
- 部分卡片会追问喜欢／不喜欢的具体原因；
- 支持撤销上一步；
- 完成后比较三个方向的喜欢率并生成视觉偏好报告；
- 结果可复制给设计方；
- 结果只保存在当前浏览器。
